#  Data Jobs Market Analysis

**Tools:** SQL (SQLite · Window Functions · CTEs) · Power BI · DAX  
**Period:** 2020–2024 | **Countries:** 14 | **Author:** Dimitrios Antoniou

---

##  Project Overview

Analysis of the global data jobs market focusing on salary trends, experience levels, 
remote work distribution and country-based comparisons for Data roles 
(Data Analyst, Data Scientist, Data Engineer and more).

---

##  Key Insights

- **US leads** in Data Analyst salaries (~$92K avg)
- **Executive-level roles earn 2x more** than entry-level positions
- **Remote work accounts for ~35%** of positions globally
- **Strong market growth between 2021–2023**, followed by stabilization
- **Senior Data Analyst roles command a 30–40% salary premium** over mid-level
