# Data Jobs Market Analysis
Portfolio project analyzing the global data jobs market (2020–2024) using SQL and Power BI.
## Project Overview
This project explores:
- salary trends
- experience levels
- country-based comparisons
- remote vs hybrid vs onsite work modes
- year-over-year salary growth
## Tools Used
- SQL
- Power BI
- CSV / Data Cleaning
- Window Functions
- CTEs
## Files Included
- `ds_salaries_2024.csv` → main dataset
- `SQL_Queries.sql` → analytical SQL queries
- `Data_Jobs_Market_Analysis.pbix` → interactive dashboard
- `Data_Jobs_Market_Analysis.pdf` → dashboard export
- `salary_tiers.csv`
- `above_avg_roles.csv`
- `yoy_growth.csv`
## Dashboard Sections
1. Market Overview
2. Data Analyst Insights
3. Advanced SQL Insights
## Key Insights
- salary growth trends from 2020 to 2024
- highest paying roles
- countries with strongest demand
- work mode distribution
